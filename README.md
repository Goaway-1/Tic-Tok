> ## 사용한 기술 및 배운점
  - 프리펩
  - 로고 제작
  - 기본적인 게임 돌아가는 방식
  - 구글 애드몹 사용법
    - 배너만 구현
  - PlayerFabs 사용법
    - 모든 데이터를 PlayerFabs를 사용
    - 보안에 취약
  - PlayStore 출시 관련 필요한 자료들
    - 아이콘, 출시 과정 등등
    - 출시 검사 오랜 시간 소유 (약 1주일)

> ## 개선점
  - 후반으로 갈수록 너무 높은 난이도
  - 최적화가 이루어져있지 않다.
  - 스크립트의 복잡성과 유일성
  - 지형이 겹치는 경우가 다반사
